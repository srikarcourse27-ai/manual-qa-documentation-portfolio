# Manual QA Documentation Portfolio

## Overview
This repository contains manual QA testing documents created for the SauceDemo e-commerce website.

## Project Scope
The project demonstrates the complete Software Testing Life Cycle (STLC) documentation for a sample web application.

## Documents Included

- Test Plan
- Requirements Document
- Test Cases
- Bug Report Log
- Requirement Traceability Matrix (RTM)
- Test Summary Report
